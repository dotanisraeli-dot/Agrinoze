# SensAItion Agronomist Simulator v2.1

## Quick Start

### Step 1: Install dependencies
```bash
npm install
```

### Step 2: Run the simulator
```bash
npm run dev
```

This will automatically open the simulator in Chrome. If it doesn't, go to `http://localhost:5173`

## What You Can Do

- **Start System**: Initialize with a soil type
- **Run Sim**: Run the algorithm step-by-step or continuously
- **Speed Control**: Adjust simulation speed (0.5x to 5x)
- **Manual Override**: Take control in Semi-Auto or Full Manual mode
- **Upload Data**: Load real sensor data from CSV/TXT files
- **Export Log**: Download the algorithm decision log

## Modes

- **Automatic**: Engine in full control
- **Semi-Auto**: Algorithm continues with your manual baseline
- **Full Manual**: Algorithm paused, you control irrigation only
- **Frozen**: Tap closed (test scenario)
